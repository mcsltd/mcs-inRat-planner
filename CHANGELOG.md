# Changelog

## [0.0.3-beta] - 2026-01-12

### Добавлено

* добавлена индексация сделанных вручную записей ЭКГ в таблицу "Записи"
* добавлена опция в настройках расписания - "Не использовать регистрацию по расписанию"
* добавлена индикация состояния расписания в таблице "Расписания" - `"Истекло"`, `"Не запланировано"`

### Изменено

* новый формат именования записей сигналов ЭКГ: `{Эксперимент}_{Объект}_{yyyy}-{mm}-{dd}_{hh}-{mm}-{ss}.{format}`

### Исправлено

* исправлено время записи сигналов, теперь запись, сделанная по расписанию, имеет фиксированное время определяемое параметром `Длительность`
